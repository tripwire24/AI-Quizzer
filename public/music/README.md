# Background Music

Drop 5 short `.mp3` clips (≈ 30 seconds each) into this folder.  
They'll play randomly on the **host screen** between rounds.

**Naming convention:**
```
track1.mp3
track2.mp3
track3.mp3
track4.mp3
track5.mp3
```

**Good sources for royalty-free quiz/game music:**
- [Pixabay Music](https://pixabay.com/music/) — free, no attribution required
- [Uppbeat](https://uppbeat.io/) — free tier available
- [Mixkit](https://mixkit.co/free-stock-music/) — free, no sign-up

Look for: "quiz show", "thinking music", "game show", "trivia" tags.  
Keep clips ≈ 20-30 seconds. They loop automatically.

**If no files are present, the game works normally — music is optional.**
